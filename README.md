## Setup
- Add `nuxt-leaflet` dependency using yarn or npm to your project
- Add `nuxt-leaflet` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-leaflet',

    // With options
    ['nuxt-leaflet', { /* module options */ }],
 ]
}
```
